Pretérito Imperfecto
de Subjuntivo

3 ejercicios de producción · Corrección inmediata

Regla de formación

hablar → hablaron
→ hablara / hablaras…
comer → comieron
→ comiera / comieras…
vivir → vivieron
→ viviera / vivieras…
tener → tuvieron
→ tuviera / tuvieras…
ir/ser → fueron
→ fuera / fueras…
poder → pudieron
→ pudiera / pudieras…
Truco: 3ª persona plural del indefinido → quita -ron → añade -ra / -ras / -ra / -ramos / -rais / -ran. Si el indefinido es irregular, el subjuntivo hereda esa irregularidad.
